Membuat halaman web modern dan responsif!

Tujuan dari proyek ini adalah untuk membuat halaman web bebas (contoh: profil perusahaan, halaman tur & travel, halaman jual-beli, halaman profil tim atau sebagainya) tanpa arahan dan atau instruksi apapun.

Untuk dinyatakan “PASS”, Halaman Web Anda harus memiliki 6 komponen: Header, Bagian utama, Bagian tambahan, Footer, Setidaknya satu animasi, dan Desain responsif untuk smartphone dan tablet

Pastikan untuk :

Melihat slide library jika ada yang lupa cara penulisan syntax

Bagi tugas secara merata (misalnya si A bikin header, si B bikin body, si C bikin footer, dstnya)

Referensi tambahan:

Color code: flatuicolors.com

Berbagai macam animasi CSS: CSS Animations

Untuk mencari gambar ilustrasi: undraw.co
